Anti-Plagiarism

Description

Our platform will check your code for plagiarism by using Machine Learning and Rabin-Karp algorithm.

First you need to register. If you have an account then login and gain access to our checker. Both processes requires you to enter e-mail and password. As soon as you do that you'll get token and an access to checker.

The form allows you to choose programming language which you used to write code. The list of available languages to check consists of Python, C++ and Java. After that you just need to upload file and press submit button.
