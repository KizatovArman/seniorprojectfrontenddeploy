Anti-Plagiarism

Description

Our platform will check your code for plagiarism by using Machine Learning and Rabin-Karp algorithm.

First you need to register. Please use email provided by kbtu. If you have an account then login and gain access to our checker. Both processes requires you to enter e-mail and password. As soon as you do that you'll get token and an access to checker.

You just need to upload file and press submit button. The list of available languages to check consists of Python, C++ and Java, please choose carefully. If request is successfull then you will see your plagiarism percentage, otherwise you will be informed about error. If you upload c++ file you can alternatively check by our Machine Learning model.
